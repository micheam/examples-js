# parcel project template with vuejs + vuetify

## Requirements

- yarn or npm

## Usage

Init this project:

```sh
yarn install
```

or

```sh
npm install
```

Serve via dev server:

```sh
yarn run parcel index.html ./static/*.* --open
```

or

```sh
npx parcel -- index.html ./static/*.* --open
```
